Fraud Detection System is an end-to-end machine learning platform designed to detect, explain, and monitor fraudulent transactions in real time.

Key Features
Machine Learning Pipeline: Complete data simulation, preprocessing, training, and evaluation models to identify fraudulent patterns.

Explainable AI (XAI): Built-in model interpretability to provide clear explanations behind flagged transactions.

AI Assistance: An integrated chatbot interface to assist users with transaction queries and system insights.

Web Interface: Interactive dashboard (HTML/JS/CSS) for user interaction and visualization.

Production-Ready & Monitored: Fully containerized with Docker and configured for real-time metrics monitoring via Prometheus.
